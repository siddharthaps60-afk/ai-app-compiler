# AI App Compiler

A multi-stage AI-powered software generation system that converts natural language requirements into validated UI, API, Database, and Authentication schemas.

## Features

- Intent Extraction
- System Design
- Schema Generation
- Validation Engine
- Repair Engine
- Runtime Generation

## Tech Stack

- FastAPI
- React
- OpenAI API
- Pydantic

## Pipeline

1. Intent Extraction
2. System Design
3. Schema Generation
4. Validation
5. Repair
6. Runtime Generation
